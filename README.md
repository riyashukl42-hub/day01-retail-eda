# Day 01 — Supermarket Sales EDA

## Problem statement
What are the key sales patterns across branches, product lines, and customer types?

## Dataset
- Source: Kaggle — Supermarket Sales
- 1,000 transactions across 3 branches, Jan–Mar 2019

## Key findings
- Branch C leads in total revenue
- Food & Beverages is the top product line
- Peak hours: 11am, 1pm, 7pm
- Members vs Normal customers split ~50/50

## Tools used
Python · Pandas · Matplotlib · Seaborn · VS Code
